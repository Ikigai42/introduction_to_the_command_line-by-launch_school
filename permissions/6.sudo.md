### 6. Practice using sudo to log in as root:

`$ cd`

`$ sudo su`

`Password:`

`root@host:/home/ubuntu#`

## Exit out of the root user session:

`root@host:/home/ubuntu# exit`

## Log in as root in root's home directory:

`$ sudo su -`

## Run a command as root:

`$ sudo ls /root`
